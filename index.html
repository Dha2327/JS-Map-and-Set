Q2-Problem: You are given a string (STR) of length N, you have to print the count of all alphabet.(using maps) 
Input:
                abcadeecfb
                
                Output:
                a=2
                b=2
                c=2
                d=1
                e=2
                f=1


Ans-
var string = "abcadeecfb";
                    const myMap = new Map();
                    for (var alphabet of string) {
                      if (myMap.has(alphabet)) {
                        myMap.set(alphabet, myMap.get(alphabet) + 1);
                      } else {
                        myMap.set(alphabet, 1);
                      }
                    }
                    console.log(myMap);
